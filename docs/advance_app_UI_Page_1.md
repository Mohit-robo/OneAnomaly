# MASTER UI PROMPT — PAGE 1

Design a modern multi-page industrial AI platform dashboard inspired by the structural UX philosophy of the [Roboflow Platform](https://roboflow.com/?utm_source=chatgpt.com), but NOT a clone.

The application is a high-end computer vision workflow platform used for anomaly detection, segmentation, preprocessing pipelines, TensorRT optimization, memory bank workflows, and industrial inspection systems.

The UI should feel:

* technical
* premium
* focused
* modular
* production-grade
* visually calm
* built for engineers, not consumers

Maintain the EXISTING application colors.
Do NOT copy Roboflow’s colors.
Only borrow:

* layout hierarchy
* spacing rhythm
* interaction philosophy
* panel structure
* workflow organization
* sidebar navigation behavior
* information density balance

The visual style should combine:

* Roboflow’s clean workflow structure
* linear.app level polish
* modern MLOps dashboard aesthetics
* subtle glassmorphism
* soft depth
* minimal gradients
* precision-engineering feel

---

# OVERALL LAYOUT

The application is a full-screen desktop-first SPA with:

* fixed left workflow sidebar
* top contextual toolbar
* main workspace canvas
* optional right inspector panel
* modular card system

The experience should feel like:
“a visual AI pipeline operating system.”

Use:

* rounded corners (12–16px)
* subtle shadows
* thin borders
* layered dark surfaces
* floating cards
* smooth hover transitions
* compact typography
* generous whitespace
* sharp visual hierarchy

Avoid:

* consumer-app aesthetics
* oversized padding
* playful UI
* bright gradients
* neumorphism
* clutter

---

# SIDEBAR (MOST IMPORTANT)

The left sidebar should strongly resemble the workflow-oriented navigation philosophy of Roboflow.

Fixed vertical sidebar with:

* logo at top
* workspace/project switcher
* collapsible workflow stages
* stage progress indicators
* nested subsections
* active stage highlighting

The sidebar is NOT a simple nav menu.
It should feel like:
“a guided ML pipeline builder.”

Sidebar behavior:

* expandable sections
* hover glow
* active vertical accent line
* animated transitions
* sticky positioning
* scrollable if overflow

Example structure styling:

* section title
* nested pipeline steps
* subtle icons
* current stage emphasized
* completed stages with checkmarks
* future stages dimmed

Visual hierarchy:

1. Current active step
2. Completed steps
3. Available steps
4. Locked/future steps

The sidebar should feel intelligent and workflow-aware.
